Build a scalable offline church lyrics presentation system using Flask, SQLAlchemy, Flask-SocketIO, and SQLite.

The system runs locally on macOS and integrates with OBS using a Browser Source. The OBS Browser Source loads /output and must have a transparent background. OBS handles video backgrounds вЂ” the web app renders only lyrics.
